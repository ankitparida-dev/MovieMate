# 🎬 MovieMate

MovieMate is a fully interactive and responsive **React.js movie & TV show discovery platform**.  
It provides a smooth and modern UI to explore, search, and view movies and shows with dynamic features and real-time interactivity.

---

## ✨ Features

- 🔎 **Live Movie Search** – Dynamic search with instant filtering  
- ⭐ **Add to Watchlist** – Save and manage favorite movies (React state + JSON Server)  
- ❤️ **Favorites / Like System** – Mark movies you love  
- 📜 **Watch History** – Automatically tracks viewed movies  
- 🎥 **Movie Details Page** – Detailed info with dynamic routing  
- 🎞️ **Spotlight Carousel** – Interactive hero movie slider  
- 🎬 **Browse Movies & Series** – Separate pages for Movies, TV Shows  
- 👤 **User Login & Registration** – Fully functional using JSON Server  
- 📱 **Fully Responsive** – Optimized for desktop, tablet, and mobile  
- ⚡ **Fast & Smooth UI** – Built with React hooks and modular components  

---

## 🧩 Tech Stack

| Technology | Purpose |
|-----------|---------|
| **React.js (Vite)** | Main UI, components, routing & interactivity |
| **CSS Modules** | Component-based styling |
| **JavaScript (ES6+)** | Logic & API handling |
| **JSON Server** | Local mock backend for Login, Register, Watchlist |
| **TMDB API (optional)** | Movie & TV show data |
| **Fetch** | API communication |

---

## 🧠 Learning Goals

This project helps in understanding and applying:

- Modular and reusable **React components**
- State management using `useState` and `useEffect`
- API handling using **Fetch** or **Axios**
- Designing **responsive layouts** with CSS Modules
- Client-side navigation and page switching
- Building mock backend features with **JSON Server**
